# SymLang Syntax Highlighting

This extension provides syntax highlighting for SymLang (.sir) files.

## Features

- Syntax highlighting for keywords, types, identifiers, literals, and operators.
- Support for both line and block comments.
- Automatic bracket matching and closing.

## About SymLang

SymLang (also known as SymIR) is a CFG-based symbolic IR designed for program synthesis, symbolic execution, and constraint generation for SMT solvers.
